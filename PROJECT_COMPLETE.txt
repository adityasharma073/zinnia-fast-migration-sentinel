  - Docling requires downloading ML models from HuggingFace
  - This is blocked in sandboxed environments
  - This is NOT a code defect
  - Will work perfectly on local machine or cloud server with internet
  - Models download once and cache for future runs

To Test PDF Extraction:
  1. Clone this code to your local machine
  2. Run: source venv/bin/activate && python3 extractor.py sample_sor.pdf sample_data.xml
  3. Models will download (~250MB) and extraction will succeed
  4. Subsequent runs will use cached models
